# v0.3.0

This release turns the kit into a more adoptable early-stage open-source project.

## Highlights

- Unified `mpmg` CLI for initialization, validation, audit, doctor checks, worker task generation, public export, and handoff cleanup planning.
- `doctor` command for environment and governance health checks.
- Non-mutating handoff cleanup planner.
- Artifact validator for worker task briefs and review identity metadata.
- Synthetic multi-agent review and public-export examples.
- Roadmap, changelog, quickstart, adoption matrix, and release checklist.
- Stronger CI smoke tests for new workflows.

## Safety stance

The release contains synthetic examples and sanitized process guidance only. It does not include private project paths, credentials, targets, run artifacts, or sensitive logs.
