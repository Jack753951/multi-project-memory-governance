## Memory boundary

Global durable memory is not the project database. Treat it only as compact user preferences and cross-project signposts. Repo handoff files are the current engineering truth for this synthetic demo.

## External worker rule

External workers do not inherit the coordinator's chat history or durable memory. Every worker task must list required context reads and allowed scope.
