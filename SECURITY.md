# Security Policy

This repository is a public governance template. Please do not open issues or PRs containing private credentials, tokens, cookies, target details, raw security evidence, customer data, or personal paths.

## Reporting sensitive leaks

If you find sensitive material in this repository, open a minimal GitHub security advisory or contact the maintainer privately. Include only the file path and line range, not copied secrets.

## Supported versions

The `master` branch is the supported public template branch.
